# 🍽️ Swiggy Sales Analysis Dashboard  

## 📌 Project Overview  
The **Swiggy Sales Analysis Project** is an end-to-end data analytics project focused on understanding customer ordering behavior, sales trends, restaurant performance, and regional demand patterns using Swiggy food delivery data.  

This project transforms raw food delivery data into actionable business insights through **data cleaning, exploratory data analysis (EDA), KPI tracking, and interactive visual storytelling**.  

The goal is to help businesses identify:
- High-performing cities & states  
- Customer food preferences  
- Seasonal sales trends  
- Revenue-driving categories  
- Customer satisfaction patterns  

---

# 🎯 Business Problem Statement  
Food delivery platforms generate massive transactional data daily. Businesses often struggle to identify:
- Which cities generate the highest revenue  
- How customer ratings impact sales  
- Which food categories perform best  
- Seasonal or weekly order fluctuations  
- Average customer spending patterns  

This project solves these challenges using analytical dashboards and KPI-driven insights.

---

# 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| 💰 Total Sales | Overall revenue generated from food orders |
| 📦 Total Orders | Total number of food orders received |
| ⭐ Average Rating | Customer satisfaction level |
| 🧾 Average Order Value | Revenue generated per order |
| 🗳️ Ratings Count | Total customer reviews received |

# 🔍 Key Business Insights  

## 📌 Sales Insights
- Sales fluctuate significantly across months, showing seasonal demand patterns.
- Certain weeks contribute exceptionally high revenue, indicating festival or promotional impact.
- Daily trend analysis shows peak ordering behavior during weekends.

## 📌 Customer Insights
- Higher-rated restaurants consistently receive more orders.
- Average customer ratings indicate overall strong customer satisfaction.

## 📌 Geographic Insights
- Metro cities dominate total revenue contribution.
- Few states contribute disproportionately high sales compared to others.

## 📌 Food Preference Insights
- Clear preference patterns exist between Veg and Non-Veg categories.
- One category contributes significantly higher revenue, helping businesses optimize inventory and marketing.
